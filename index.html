Elementos de bloque y de línea

Una explicación de porqué algunos elementos se comportan de manera distinta

el párrafo es un elemento de bloque y ocupa todo el ancho del navegador, por lo que el siguiente elemento va debajo.

el div es otro elemento de bloque y ocupa todo el ancho

Los enlaces, al igual que las imágenes son elementos de línea, van seguidos como las letras de una línea

para que la primera imágen se comporte como un elemento de bloque, la he metido como si fuera un párrafo.
